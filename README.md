PR 6 File Operator Project
Personal Journal Manager

Project Functions
User can add a new journal entry
User can see all saved entries
User can search entries using keyword or date
User can delete all entries
Menu driven program is used

Class Used
JournalManager class is used to manage file operations

File Handling
journal.txt file is used to store entries
Append mode is used for adding data
Read mode is used for viewing and searching

Exception Handling
File not found error is handled
Permission error is handled
Invalid menu input is also handled

How Program Works
Menu is displayed to the user
User selects an option
Input is taken from the user
Operation is performed
Program runs until exit option is selected

Assumptions
User enters correct input
Single journal file is used

Conclusion
This project helped me understand file handling and OOP concepts in Python. It is simple and easy to understand.
